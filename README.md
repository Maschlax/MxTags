# MxTags
A simple PlayerTags Plugin (Player suffixes)

## Features
- Database Connection through config settings
- Debug mode for getting the exact error for something.
- ColorFormatter, returns a colorful minecraft useable string with hexcode input in this format: <#ffffff>
- Tags have support for RGB colors in the ColorFormatter format
- /tag command to select a tag by ID (later through GUI)
- /tagmanager : Show Help message
- /tm create <name> <slot>
- /tm delete <tagID>
- /tm list : List all tags
